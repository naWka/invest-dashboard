## Часть 3. Алгоритм покупки

### Шаг 1. Рассчитать текущую стоимость каждой позиции

```
Стоимость_SWRD = кол-во_SWRD × текущая_цена_SWRD
Стоимость_EIMI = кол-во_EIMI × текущая_цена_EIMI
Стоимость_USSC = кол-во_USSC × текущая_цена_USSC
Портфель_текущий = сумма всех трёх
```

### Шаг 2. Рассчитать целевую стоимость

```
Портфель_будущий = Портфель_текущий + Сумма_покупки
Цель_SWRD = Портфель_будущий × 0.70
Цель_EIMI = Портфель_будущий × 0.15
Цель_USSC = Портфель_будущий × 0.15
```

### Шаг 3. Рассчитать дефицит

```
Дефицит = Цель − Стоимость
```

### Шаг 4. Распределить сумму покупки

Покупаем в порядке наибольшего дефицита. `Штук = floor(Дефицит / цена)`. Остаток → кэш.

### Шаг 5. Проверка

Доля каждого ETF не отклоняется от целевой более чем на ±5 п.п.

### Особые правила

- **< $100**: один ETF с наибольшим дефицитом
- **$100–200**: 1–2 ETF с наибольшим дефицитом
- **Просадка >15% от пика**: всё пополнение → в просевший ETF
